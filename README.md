# 🎨 ALAF Client (Frontend)

> **대학 분실물 통합 관리 시스템 - 웹**

이 디렉토리는 ALAF 프로젝트의 **프론트엔드(React) 웹 서비스** 소스 코드를 담고 있습니다.  
학생들이 언제 어디서나 PC 및 모바일 기기를 통해 교내 분실물을 조회하고, 회수 신청 및 알림 설정을 할 수 있도록 직관적인 사용자 경험(UX)을 제공합니다.

*(※ 라즈베리파이 연동 키오스크(Kiosk) 모듈은 로컬 환경에서 별도로 구동 및 관리됩니다.)*

---

## 🚧 개발 진행 상황 (Current Status)

**현재 단계: 클라우드 서버 연동 및 주요 기능 구현 완료 (Production Ready)**

백엔드 클라우드 서버와의 API 연동을 마치고 핵심 비즈니스 로직(조회, 신청, 승인, 알림 등)이 모두 구현된 상태입니다.

## 📂 프로젝트 구조 (Structure)

```bash
src/
├── context/             # ⚡️ 전역 상태 관리 (데이터 공급소)
│   ├── ItemContext.js   # 분실물 데이터 CRUD 및 API 호출 로직
│   └── UserContext.js   # 유저 로그인/인증 로직 (현재 Mock Data)
│
├── ui/                  # 🖼️ 화면 컴포넌트
│   └── web/             # [PC/Mobile] 일반 웹 브라우저용 페이지
│       ├── WebHome.jsx      # 메인: 검색, 카테고리 필터
│       ├── WebRegister.jsx  # 등록: 정보 입력 및 이미지 업로드
│       └── ...
│
├── App.js               # 🚦 라우팅 설정
└── App.css              # 🎨 전역 스타일 초기화 (Reset CSS)
```

## 🛠 주요 기능 (Key Features)

### 🖥️ 웹 (Web Interface)
일반 사용자가 PC나 모바일로 접속하는 화면입니다. (`/`)

* **🔍 통합 검색 및 필터링**
    * 물건명, 습득 장소 등으로 키워드 검색 가능
    * 카테고리(전자기기, 지갑, 의류 등)별 모아보기 지원
    * 최신순 정렬 기능
* **📄 상세 정보 확인**
    * 습득된 물건의 사진, 습득 일시, 보관 장소 확인
    * 분실물 회수 신청 시, 증빙 자료(사진, 설명 등) 업로드 기능 제공
    * 회수 신청 즉시 물건이 '48시간 잠금' 처리되어 중복 신청 방지
* **👤 회원 서비스**
    * **로그인/회원가입**: 기본 인증 절차 UI 및 유효성 검사 로직
    * **이메일 알림 구독**: 마이페이지에서 본인이 잃어버린 물건의 카테고리(예: 가방, 카드)를 체크해 두면, 해당 물건이 등록될 때 구글 이메일로 즉시 알림을 받을 수 있습니다.
    * **권한 분기**: 로그인한 계정이 관리자(ADMIN)일 경우, 마이페이지에 '관리자 전용 대시보드' 메뉴가 활성화되어 유저들의 회수 요청을 심사(승인/거절)할 수 있습니다.
